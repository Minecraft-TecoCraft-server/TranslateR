### 介绍

<br/>一个能在游戏内对中文与其它语言进行实时翻译的插件
<br/>使用`!!t <语句>`即可对`语句`进行以下形式的翻译：
|支持的翻译形式 |
| :---: |
| 汉语→英语 |
| 英语→汉语 |
